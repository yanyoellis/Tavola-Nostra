# Tavola Nostra

Интерактивный сайт итальянского ресторана: пользователь сидит за столом, а блюда и предметы работают как элементы навигации.

## Возможности

- интерактивная ресторанная сцена;
- страницы меню, истории ресторана и корзины;
- 18 позиций в 6 категориях;
- сохранение корзины в `localStorage`;
- адаптивная версия для планшетов и телефонов;
- оригинальная серия из 10 фотореалистичных изображений.

## Запуск

Требуется Node.js 20 или новее.

```bash
npm install
npm run dev
```

Production-сборка:

```bash
npm run build
```

## Стек

React, Vite, CSS, localStorage.
