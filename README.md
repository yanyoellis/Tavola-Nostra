# Tavola Nostra

Интерактивный сайт итальянского ресторана: пользователь будто сидит за деревянным столом у окна, а блюда и предметы на столе работают как элементы интерфейса.

## Возможности

- интерактивная ресторанная сцена на главной странице;
- страницы меню, истории ресторана, корзины и UX case study;
- UX case study с переключением языка: English, Русский, Polski, Українська;
- 18 позиций в 6 категориях;
- сохранение корзины в `localStorage`;
- адаптивная версия для планшетов и телефонов;
- оригинальная серия из 10 фотореалистичных изображений.

## Страницы

- `/` — интерактивный стол ресторана;
- `/menu` — полное меню;
- `/about` — история ресторана;
- `/cart` — корзина в виде ресторанного чека;
- `/case-study` — UX case study проекта.

## Запуск

Требуется Node.js 20 или новее.

```bash
npm install
npm run dev
```

Production-сборка:

```bash
npm run build
```

## Стек

React, Vite, CSS, localStorage.
